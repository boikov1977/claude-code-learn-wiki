💻 Установка в терминале

Основной способ — через npm. Одна команда:

npm i -g @anthropic-ai/claude-code

После установки проверь версию:

claude --version

Если npm выдаёт ошибку прав

Не используй sudo npm install. Вместо этого настрой npm чтобы ставить пакеты в домашнюю директорию:

mkdir -p ~/.npm-global npm config set prefix '~/.npm-global' echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc source ~/.bashrc

После этого npm i -g будет работать без sudo.

Обновление

Чтобы обновить до последней версии:

npm update -g @anthropic-ai/claude-code

🖥️ Установка в IDE

VS Code

  1. Открой VS Code
  2. Extensions (Ctrl+Shift+X)
  3. Найди «Claude Code»
  4. Установи расширение

Или через CLI: code --install-extension anthropic.claude-code

JetBrains

  1. Открой Settings → Plugins
  2. Marketplace → найди «Claude Code»
  3. Установи и перезапусти IDE

Поддерживается: IntelliJ, PyCharm, WebStorm, GoLand и др.

Claude Desktop

  1. Скачай с claude.ai/download
  2. Установи и запусти
  3. Войди в аккаунт Anthropic

Claude Code доступен как встроенный режим в десктопном приложении.

Веб (claude.ai)

  1. Открой claude.ai
  2. Войди в аккаунт
  3. Выбери Claude Code в интерфейсе

Облачный режим: работает без установки, но с ограничениями.

🔑 Авторизация

Claude Code поддерживает два способа авторизации:

✅ Claude Pro / Max (рекомендуется)

Что это: Подписка Anthropic ($20/мес Pro, $100/мес Max).

Как подключить: Запусти claude — он предложит войти через браузер. Авторизуйся через аккаунт Anthropic.

Преимущества:

  • Не нужен API-ключ
  • Включены все модели (Opus, Sonnet)
  • Проще настройка

🔐 API Key

Что это: Ключ API Anthropic для платного использования.

Как подключить:

export ANTHROPIC_API_KEY=sk-ant-...

Когда использовать:

  • Нет подписки Pro/Max
  • CI/CD пайплайны
  • Автоматизация
💡 Совет: Если ты только начинаешь — используй Claude Pro ($20/мес). Это самый простой способ: один клик, никаких ключей. API-ключ понадобится, если захочешь автоматизировать или использовать в CI/CD.

✅ Проверка установки

После установки и авторизации проверь, что всё работает:

1

Проверь версию

claude --version

Должна показать текущую версию, например: 1.0.24

2

Запусти Claude

claude

Должен открыться интерфейс Claude Code в терминале с промптом для ввода.

3

Отправь тестовый промпт

привет, как дела? скажи свою версию

Claude должен ответить и показать, что он работает.

⚙️ Базовая настройка

После установки стоит проверить несколько базовых настроек:

Просмотр текущих настроек

claude config list

Установка модели по умолчанию

# Использовать последнюю модель Sonnet (по умолчанию) claude config set model claude-sonnet-4-20250514 # Или Opus для сложных задач claude config set model claude-opus-4-20250514

Настройка разрешений

# Разрешить Write/Edit без подтверждения (осторожно!) claude config set autoApprove Write Edit # Или наоборот — всегда спрашивать claude config set autoApprove ""

⚠️ Не отключай разрешения без понимания

Режим bypassPermissions отключает все подтверждения. Claude будет писать файлы и запускать команды без вопросов. Это удобно, но опасно: можно случайно удалить файл или запустить опасную команду. Начни со стандартных настроек и расширяй по мере необходимости.

CLAUDE.md — память проекта

Для каждого проекта стоит создать CLAUDE.md в корне — это инструкции для Claude, которые он читает при каждом запуске. Туда пишут:

  • Какой стек используется
  • Как запускать тесты
  • Какие конвенции по стилю кода
  • Что можно и нельзя делать
# CLAUDE.md ## Проект Python-бот для Telegram, FastAPI, SQLAlchemy ## Запуск тестов pytest tests/ -v ## Стиль - Black для форматирования - ruff для линтера - Type hints обязательны ## Не трогать - alembic/migrations/ (только через alembic)

🔗 Что дальше

Установка завершена. Вот рекомендуемый порядок действий:

  1. Отправь первый промпт — попробуй простую задачу. Подробнее: Твой первый промпт.
  2. Создай CLAUDE.md — опиши свой проект для Claude.
  3. Попробуй Explore → Plan → Code → Commit — полный цикл работы. Подробнее: Explore → Plan → Code → Commit.
  4. Настрой permissions — убери лишние подтверждения для routine-операций.

Итого: Установка — одна команда npm i -g. Авторизация — через Pro/Max или API key. Настройка — claude config и CLAUDE.md. Всё готово к работе.
