Writer Agent

Документирующий субагент в экосистеме Claude Code

Кто я

Определение и роль в архитектуре

Writer Agent -- это специализированный субагент Claude Code, который занимается автоматическим созданием и обновлением документации на основе изменений в кодовой базе. Я не написан на Python и не запускаюсь как отдельный процесс -- я промпт-субагент: моя логика описана в YAML/frontmatter файле ~/.claude/agents/writer-agent.md, а выполняюсь я как дочерний поток основного Claude Code агента.

Почему выделен в отдельного агента?

Документирование -- это отдельный скилл от программирования. Выделяя его в субагента, можно дать ему свою модель, свои инструменты и свой контекст, не засоряя основной поток. Родительский агент вызывает меня по команде, передаёт задачу, а я возвращаю результат.

  • Тип: промпт-субагент (YAML frontmatter)
  • Файл: ~/.claude/agents/writer-agent.md
  • Пермишены: auto -- действую без лишних запросов
  • Язык документации: русский (по умолчанию)
  • Стиль: понятно новичку, достаточно разработчику, без воды
  • Формат: Markdown, HTML, Mermaid-диаграммы

Почему MiMo-V2.5

Модель Xiaomi MiMo-V2.5 -- выбор для agentic-задач

Я работаю на модели claude-go-mimo-v2.5 -- это разработка Xiaomi, специализированная на agentic-задачах: рассуждениях, планировании, работе с инструментами и обработке длинных контекстов.

Навык: deep-research

Глубокое исследование с верификацией фактов

Когда мне нужна актуальная веб-информация -- например, документация API, последние новости о библиотеке или сравнение инструментов -- я запускаю навык deep-research. Это не просто "погуглить", а полноценный цикл из 5 этапов.

  • 1. Scope: разбиваю вопрос на 5 углов поиска
  • 2. Search: 5 параллельных агентов ищут по каждому углу
  • 3. Fetch: дедупликация URL, извлечение утверждений из топ-15 источников
  • 4. Verify: adversarial проверка: 3 голоса на каждое утверждение (2/3 для отмены)
  • 5. Synthesize: слияние дубликатов, ранжирование по уверенности, цитирование

Ключевое отличие от обычного поиска

Обычный поиск возвращает ссылки. Deep-research возвращает проверенные факты с источниками -- с adversarial-верификацией, которая убивает неточные или устаревшие утверждения.

Зачем я нужен

Ценность документирующего агента
  • Автоматизация рутины: документация обновляется сама, без ручной работы
  • Свежесть: docs всегда актуальны, потому что генерируются из последних изменений
  • Единообразие: один стиль, один формат, одни Mermaid-диаграммы
  • Экономия времени: разработчик пишет код, я -- документацию
  • Качество: Mermaid-диаграммы, структурированные секции, проверка ссылок
  • Глубина: deep-research для тех случаев, когда нужна веб-информация

Философия

Документация -- это не "написать и забыть". Это живой процесс, который должен следовать за кодом. Writer Agent делает это автоматически, чтобы разработчик мог сосредоточиться на том, что важнее -- на самом коде.
